# Brandbook и дизайн сайта

## Характер бренда

- Аккуратный
- Локальный
- Уверенный
- Непоказной
- Профессиональный

## Визуальное решение

Бренд должен существовать в двух темах:

- Темная тема как базовая. Она лучше передает фактуру, мастерскую эстетику и совпадает с референсами.
- Светлая тема как альтернативная. Она усиливает ощущение чистоты, понятности и сервиса.

Это не два разных бренда. Это одна структура, одна сетка, одна типографика и один UX.

## Палитра

- Базовый темный: графит, угольный, дымчато-серый.
- Базовый светлый: молочный, теплый серо-бежевый, бумажный.
- Акцент: теплый латунно-песочный, без ощущения золота.
- Цвет CTA в светлой теме: темный графит.
- Цвет CTA в темной теме: теплый песочный.

## Фото и графика

- Реальные изделия вместо стоков.
- Крупные планы кожи, фактуры, швов, подошвы, фурнитуры.
- Аккуратные до/после.
- Живая мастерская, но без визуального шума.
- Простые иконки и минимум декоративной графики.

## UX-принципы

- Первый экран должен сразу объяснять услугу, город и следующий шаг.
- Главный CTA: отправить фото.
- Вторичные CTA: посмотреть цены, узнать как работает забор, связаться.
- На каждой коммерческой странице сначала польза и доказательства, потом детали.
- Пользователь должен оставить обращение без длинной формы и без регистрации.

## Переключение темы

- Реализовать только через CSS variables.
- Контент и URL не меняются.
- Можно использовать `prefers-color-scheme` как стартовую подсказку.
- Дать пользователю ручной toggle.
- Не дублировать DOM и не строить две версии страниц.

## Допущение

- Пользователю ближе сдержанный премиум без luxury-подачи.
