# Рекомендация в 5-7 предложениях

Для сайта Faktura / 78 оптимален SEO-first подход: компактный, очень быстрый, статически сгенерированный сайт услуг с сильными коммерческими посадочными страницами под Санкт-Петербург. Лучший стек для этого проекта — Astro + TypeScript + Content Collections + static deploy на Nginx/CDN, потому что он дает минимальный риск технических проблем с индексированием и отличный контроль над скоростью. Сайт должен позиционироваться как надежный локальный сервис с высоким качеством работы и понятным процессом обращения, а не как пафосное премиум-ателье. В структуре нужно делать ставку на страницы услуг, кейсы, цены и сроки, доставку по Санкт-Петербургу, FAQ и контакты, а не на блог. Визуально бренду подходит единая дизайн-система с темной и светлой темой: темная как базовая, светлая как альтернативная, без раздвоения идентичности. Для Яндекса критичны реальные локальные сигналы: адрес, карта, Яндекс Бизнес, региональность, schema.org и качественные коммерческие страницы. На старте важнее сделать 12-15 сильных страниц и быстрый сценарий отправки фото, чем строить сложную CMS или избыточную инфраструктуру.
